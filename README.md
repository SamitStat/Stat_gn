# Stat

A growing collection of interactive statistics visualizations - built to make
probability and inference intuitive through hands-on exploration.

## Live Demos

| Demo | Description |
|------|-------------|
| [Histogram → PDF Convergence](https://samitstat.github.io/Stat_gn/nd_to_hist.html) | Histogram converge to the true normal PDF. Mode in discrete vs. continuous. |
| [Joint Posterior 3D Slices](https://samitstat.github.io/Stat_gn/) | Interactive 3D visualization of a joint posterior distribution. |

## About

This repo is a personal space for exploring statistical concepts visually, making them interactive and fun.
Each demo is a self-contained HTML file - no installation, just click and explore.

More coming soon.
